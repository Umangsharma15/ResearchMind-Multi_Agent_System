# 🧠 ResearchMind – Multi Agent AI System

> An intelligent Multi-Agent AI system that performs research, analysis, content generation, and critical evaluation using collaborative AI agents.

---

## 🚀 Project Overview

ResearchMind is an advanced AI-powered Multi-Agent System built using Python, LangChain, and Streamlit.

The system uses multiple AI agents working together like a real team:

- 🔍 Research Agent → Collects information
- 📚 Reader Agent → Understands and summarizes content
- ✍️ Writer Agent → Generates high-quality responses
- 🧠 Critic Agent → Reviews and improves the final output

This project demonstrates how autonomous AI agents can collaborate to solve complex tasks efficiently.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Umangsharma15/ResearchMind-Multi_Agent_System.git
```

## 2️⃣ Move into Project Folder

```bash
cd ResearchMind-Multi_Agent_System
```

## 3️⃣ Create Virtual Environment

```bash
python -m venv .venv
```

## 4️⃣ Activate Environment

### Windows

```bash
.venv\Scripts\activate
```

### Linux / Mac

```bash
source .venv/bin/activate
```

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```
